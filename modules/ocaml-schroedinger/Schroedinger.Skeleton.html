Module Schroedinger.Skeleton

module Skeleton: sig .. end
val fisbone : ?start_granule:Int64.t ->
       ?headers:(string * string) list ->
       serialno:Nativeint.t ->
       format:Schroedinger.video_format -> unit -> Ogg.Stream.packet
Generate a vorbis fisbone packet with these parameters, to use in an ogg skeleton. Default value for start_granule is Int64.zero, Default value for headers is "Content-type","video/dirac"

See: http://xiph.org/ogg/doc/skeleton.html.
